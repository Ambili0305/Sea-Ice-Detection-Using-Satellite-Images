# 🌊 Sea Ice Detection using Satellite Images

## 📌 Overview

This project presents a deep learning-based approach to detect and classify sea ice concentration using satellite imagery. The model uses a U-Net Convolutional Neural Network (CNN) architecture to perform pixel-level image segmentation on Sentinel-2 satellite data.

The system automates the process of generating sea ice maps, which are useful for climate monitoring, environmental research, and maritime navigation.

---

## 🎯 Objectives

* Detect and classify sea ice concentration from satellite images
* Automate sea ice mapping using deep learning
* Improve accuracy compared to manual analysis

---

## 🧠 Methodology

* Collected Sentinel-2 satellite images and geospatial data
* Performed preprocessing (cloud masking, tiling, normalization)
* Applied data augmentation (flipping, rotation) to improve model performance
* Used U-Net CNN architecture for image segmentation
* Handled class imbalance using oversampling techniques
* Evaluated model using Accuracy and Intersection over Union (IoU)

---

## 📊 Results

* **Accuracy:** 83%
* **Mean IoU:** 0.44

The model successfully classifies sea ice into multiple concentration levels and generates segmentation masks for analysis.
